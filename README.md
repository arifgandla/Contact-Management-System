# Contact Management System

A Contact Management System built using ASP.NET Core Web API.

## Features

- User Registration
- User Login
- JWT Authentication
- CRUD Operations for Contacts
- Repository Pattern
- DTOs
- Dependency Injection
- Entity Framework Core
- PostgreSQL Database
- Swagger API Documentation

## Technologies Used

- ASP.NET Core
- C#
- Entity Framework Core
- PostgreSQL
- JWT Authentication
- Swagger
- AutoMapper
- FluentValidation
